## Dependencies

- Install CMake => https://cmake.org/

## Build

- run **./cmakegen.sh**

## Run

To generate the **image.ppm**:
- run **./build/raytracer image**


To run the tests:
- run **./build/raytracer tests**

You're all set, have fun!
